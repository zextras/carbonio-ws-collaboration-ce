# Carbonio Workstream Collaboration

This is the official repository for Carbonio Workstream Collaboration.

## Dev setup

First time install pre-commit and reuse

```shell
pip install pre-commit
```

```shell
pre-commit install
```

```shell
pipx install reuse
```
